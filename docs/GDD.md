# 游戏设计文档

## 一句话玩法

竖屏挖矿小游戏：玩家从地表向地下挖掘、收集矿石、管理氧气和背包，回到地表出售矿石并升级装备；每局开始选择一个 Roguelite 随机增益，让短局节奏产生变化。

## 目标体验

| 项目 | 决策 |
|---|---|
| 目标平台 | 抖音小游戏 |
| 屏幕方向 | 竖屏 |
| 单局时长 | 30-90 秒 |
| 核心爽点 | 贪一铲、赚一波、升一级、再下一局 |
| 原型美术 | 占位图和少量 AI 贴图优先 |

## 核心循环

```text
首页 -> 选择增益 -> 下矿 -> 挖矿收集 -> 氧气/背包制造压力
-> 回到地表出售 / 暂停返回结算 / 氧气耗尽
-> 结算金币 -> 升级装备 -> 再下一局
```

## MVP 系统

| 系统 | MVP 行为 |
|---|---|
| 挖矿 | 玩家挖相邻网格，向下探索并发现矿石 |
| 地图 | 左右无限延伸，纵向按深度推进 |
| 氧气 | 地下移动和挖掘消耗氧气；0m 地表是安全区 |
| 氧气包 | 地下偶尔出现，挖到后立即恢复一部分氧气 |
| 背包 | 矿石占用容量；矿石压缩技能让同类矿石 5 个一组 |
| 矿石经济 | 结算时矿石换金币，越深矿石价值越高 |
| 装备升级 | 矿镐、氧气瓶、背包、矿石手册 |
| Roguelite 增益 | 每局从 3 个临时增益中选择 1 个 |
| 技能 | 第一版默认启用矿石压缩，并在技能页展示 |
| 结算 | 回到 0m 可出售；暂停菜单可直接返回结算 |

## 地图规则

- 玩家初始位置是 `(0, 0)`。
- `x` 轴无限延伸，可以为负数，也可以是很大的正数。
- `y = 0` 是地表，向下递增。
- 第一版暂时保留最大生成深度，防止原型阶段出现无限深度调试问题。
- 地图按坐标确定性生成，同一局内同一坐标结果稳定，不受查询顺序影响。
- UI 只显示玩家附近区域，玩家横向移动时视口跟随玩家居中。
- 深度仍然是收益主轴；横向探索用于追矿、绕路和寻找氧气，不是无风险刷资源。

## 移动和挖掘规则

- 玩家可以向下、向左、向右挖掘相邻矿块。
- 玩家可以向上经过已经挖空的格子，用于沿原路返回。
- 玩家不能向上挖掘未挖开的泥土、石头或矿石。
- 左右没有地图边界，玩家可以持续横向探索。
- 电脑预览支持 `WASD` 和方向键，并支持长按连续移动。
- 手机预览支持滑动操作，上、下、左、右对应一次移动或挖掘。
- PC 触摸屏设备同时支持键盘和触摸输入。
- 玩家在 `0m` 地表左右移动不消耗氧气。
- 玩家从 `0m` 向下开挖第一格会消耗氧气，表示正式进入风险区。
- 玩家在地下横向移动仍消耗氧气。
- 玩家回到 `0m` 地表后，可以出售矿石进入结算，也可以继续下矿。
- 暂停菜单提供“返回结算”，用于原型阶段快速结束本局，不消耗道具。

## 氧气压力设计

| 场景 | 氧气规则 | 设计目的 |
|---|---|---|
| 0m 地表左右移动 | 不消耗 | 地表是安全区，允许调整入口位置 |
| 从 0m 向下挖 | 消耗目标矿块氧气 | 第一次下挖就是本局风险开始 |
| 地下空地移动 | 消耗移动氧气 | 让回程、绕路和继续深入都有代价 |
| 地下挖掘 | 消耗目标矿块氧气 | 用矿块类型制造节奏差异 |
| 氧气包 | 立即恢复，不占背包 | 给玩家“再贪一铲”的机会 |

- 氧气低于 20% 时，HUD 用红色提示。
- 氧气耗尽时本局结束。
- 后续可扩展“紧急返回但损失部分收益”等失败补救规则。

## 第一批增益

| 增益 | 效果 |
|---|---|
| 省氧达人 | 本局氧气消耗降低 |
| 富矿嗅觉 | 本局矿脉出现率提升 |
| 大背包 | 本局背包容量提升 |
| 快速矿镐 | 本局所有矿块更容易挖开 |
| 高价出货 | 本局矿石售价提升 |
| 深层奖金 | 每到固定深度获得额外金币 |
| 破石专家 | 本局挖石头时额外强力 |

## MVP 暂不做

- 排行榜。
- 完整广告商业化系统。
- 多人或社交竞争。
- 多角色。
- 皮肤和装扮商店。
- 长线每日任务系统。
- 复杂合成、材料、装备品质。

## MVP 验收

玩家 1 分钟内可以完成：

```text
开始下矿 -> 选择增益 -> 挖矿 -> 氧气消耗 -> 捡矿
-> 回到地表出售 / 暂停返回结算 / 氧气耗尽
-> 结算金币 -> 升级 -> 再下一局
```
