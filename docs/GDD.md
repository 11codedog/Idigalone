# 游戏设计文档

## 一句话玩法

一款竖屏挖矿小游戏：玩家下地挖矿、收集矿石、回地面卖钱、升级装备，并在每局下矿中选择 Roguelite 随机增益，让每一局都有不同手感。

## 目标体验

| 项目 | 决策 |
|---|---|
| 目标平台 | 抖音小游戏 |
| 屏幕方向 | 竖屏 |
| 单局时长 | 30-90 秒 |
| 核心爽点 | 贪一铲、赚一波、升一级、再下一局 |
| 原型美术 | 占位图形优先 |

## 核心循环

```text
首页 -> 开始下矿 -> 挖矿并收集矿石 -> 氧气/背包制造压力
-> 返回或失败 -> 卖矿结算 -> 升级装备 -> 再下一局
```

## MVP 系统

| 系统 | MVP 行为 |
|---|---|
| 挖矿 | 玩家挖相邻网格，向下探索并发现矿石。 |
| 氧气 | 移动和挖掘消耗氧气，氧气耗尽后结束本局。 |
| 氧气包 | 地下偶尔出现氧气包，挖到后立即恢复一部分氧气。 |
| 背包 | 矿石占用背包容量，背包满后推动玩家回地面。 |
| 矿石经济 | 结算时矿石换金币，越深矿石价值越高。 |
| 装备升级 | 矿镐、氧气瓶、背包、矿石售价倍率。 |
| Roguelite 增益 | 每局从 3 个随机临时增益中选择 1 个。 |
| 结算 | 展示深度、矿石、金币收益和升级入口。 |

## 移动和挖掘规则

- 玩家可以向下、向左、向右挖掘相邻矿块。
- 玩家可以向上经过已经挖空的格子，用于沿原路返回。
- 玩家不能向上挖掘未挖开的土、石头或矿石。
- 玩家主动返回地面时进入结算，不需要一步步走回 0m。
- 地下会随机生成氧气包，挖到后立即回氧，不占背包。

## 第一批增益

| 增益 | 效果 |
|---|---|
| 省氧达人 | 本局氧气消耗降低。 |
| 富矿嗅觉 | 本局稀有矿出现率提升。 |
| 大背包 | 本局背包容量提升。 |
| 快速矿镐 | 本局挖掘耗时降低。 |
| 高价出货 | 本局矿石售价提升。 |
| 深层奖金 | 每到固定深度获得额外金币。 |
| 破石专家 | 本局挖石头时额外强力。 |

## MVP 暂不做

- 排行榜。
- 完整广告商业化系统。
- 多人或社交竞争。
- 多角色。
- 皮肤和装扮商店。
- 长线每日任务系统。
- 复杂合成、材料、装备品质。

## MVP 验收

玩家 1 分钟内可以完成：

```text
开始下矿 -> 挖矿 -> 氧气消耗 -> 捡矿 -> 返回或氧气耗尽
-> 卖矿 -> 升级 -> 再下一局
```
