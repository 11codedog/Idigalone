# 游戏设计文档

## 一句话玩法

竖屏挖矿小游戏：玩家从地表向地下挖掘、收集矿石、管理氧气和背包，回到地表出售矿石并升级装备；局内完成收集目标后触发 Roguelite 三选一奖励，让短局节奏产生变化。

## 目标体验

| 项目 | 决策 |
|---|---|
| 目标平台 | 抖音小游戏 |
| 屏幕方向 | 竖屏 |
| 单局时长 | 30-90 秒 |
| 核心爽点 | 贪一铲、赚一波、升一级、再下一局 |
| 原型美术 | 少量 AI 背景/角色贴图 + 程序化矿块占位优先 |

## 核心循环

```text
首页 -> 下矿 -> 挖矿收集 -> 达成收集目标触发三选一 -> 氧气/背包制造压力
-> 回到地表出售 / 暂停返回结算 / 氧气耗尽
-> 结算金币 -> 升级装备 -> 再下一局
```

## MVP 系统

| 系统 | MVP 行为 |
|---|---|
| 挖矿 | 玩家用 360° 摇杆按时间连续推进，挖开圆形刷子范围内的土、石头和矿 |
| 地图 | 连续地形采样，横向无限延伸，纵向按深度推进 |
| 氧气 | 地下移动和挖掘消耗氧气；0m 地表是安全区 |
| 氧气包 | 地下偶尔出现，挖到后立即恢复一部分氧气 |
| 背包 | 矿石占用容量；矿石压缩技能让同类矿石 5 个一组 |
| 矿石经济 | 挖到的矿石材料单位结算为金币，越深矿石价值越高 |
| 装备升级 | 矿镐、氧气瓶、背包、矿石手册 |
| Roguelite 增益 | 局内完成收集目标后，从 3 个临时增益中选择 1 个 |
| 技能 | 第一版默认启用矿石压缩，并在技能页展示 |
| 结算 | 回到 0m 可出售；暂停菜单可直接返回结算 |

## 地图规则

- 玩家初始位置是连续坐标 `(0, 0)`。
- `x` 轴无限延伸，可以为负数，也可以是很大的正数。
- `y = 0` 是地表，向下递增，玩家位置可以是小数。
- 第一版暂时保留最大生成深度，防止原型阶段出现无限深度调试问题。
- 地图按连续坐标采样确定性生成，同一局内同一区域结果稳定，不受查询顺序影响。
- UI 只显示玩家附近区域，视口跟随玩家居中。
- 运行中视口使用高分辨率程序化视觉采样，默认至少 `160 x 220` 视觉像素，不再用大矩形块模拟地形。
- 深度仍然是收益主轴；横向探索用于追矿、绕路和寻找氧气，不是无风险刷资源。
- 第一版矿物扩展为铜、煤、锡、铁、银、金、翡翠、水晶、红宝、黑曜；浅层更快出现新颜色，中后层给高价值目标。
- 未导入贴图的矿物先使用程序化连续材质和色块展示，避免正式美术阻塞玩法验证。

## 移动和挖掘规则

- 玩家可以向任意方向连续移动和挖掘，包括向上和斜向。
- 摇杆输出 360° 方向向量，按住越久推进/挖掘距离越长。
- 挖掘使用圆形刷子，刷子范围内的连续地形采样点被清除。
- 土、石头和矿石通过硬度影响推进效率和氧气消耗。
- 左右没有地图边界，玩家可以持续横向探索。
- 电脑预览支持鼠标拖拽 360° 浮动摇杆；`WASD` 和方向键只作为离散调试 fallback。
- 手机预览支持触摸任意位置出现浮动摇杆，不再只做四方向滑动。
- PC 触摸屏设备同时支持键盘和触摸输入。
- 玩家在 `0m` 地表左右移动不消耗氧气。
- 玩家从 `0m` 向下开挖会消耗氧气，表示正式进入风险区。
- 玩家在地下横向移动仍消耗氧气。
- 玩家回到 `0m` 地表后，可以出售矿石进入结算，也可以继续下矿。
- 暂停菜单提供“返回结算”，用于原型阶段快速结束本局，不消耗道具。

## 氧气压力设计

| 场景 | 氧气规则 | 设计目的 |
|---|---|---|
| 0m 地表左右移动 | 不消耗 | 地表是安全区，允许调整入口位置 |
| 从 0m 向下挖 | 低频呼吸消耗 + 挖掘材料消耗 | 第一次下挖就是本局风险开始 |
| 地下空地移动 | 每秒少量连续消耗 | 让回程、绕路和继续深入有代价，但不因帧率暴毙 |
| 地下挖掘 | 按被清除材料单位和硬度消耗，允许小数累计 | 用材料类型制造节奏差异 |
| 氧气包 | 立即恢复，不占背包 | 给玩家“再贪一铲”的机会 |

- 当前连续地形氧气不按帧取整，避免 60fps 下每帧最低扣 1 点。
- 一级氧气瓶目标体验是浅层可玩 60 秒左右，遇到硬石/深层矿时明显更紧张。
- 氧气低于 20% 时，HUD 用红色提示。
- 氧气耗尽时本局结束。
- 后续可扩展“紧急返回但损失部分收益”等失败补救规则。

## 第一批增益

| 增益 | 效果 |
|---|---|
| 省氧达人 | 本局氧气消耗降低 |
| 富矿嗅觉 | 本局矿脉出现率提升 |
| 大背包 | 本局背包容量提升 |
| 快速矿镐 | 本局所有矿块更容易挖开 |
| 高价出货 | 本局矿石售价提升 |
| 深层奖金 | 每到固定深度获得额外金币 |
| 破石专家 | 本局挖石头时额外强力 |

## 局内三选一触发

第一版先做“奖励收集行为”，不开局白给增益。玩家在局内完成明确收集目标后，暂停操作并弹出“矿工灵感”三选一，选择后继续本局。

| 触发 | 条件 | 奖励池方向 | 目的 |
|---|---|---|---|
| 铜矿 x50 | 本局累计收集 50 单位铜矿材料 | 售价、背包、挖掘、省氧、深层奖金 | 奖励稳定采集，让玩家感觉“挖出来的 build” |

规则：

- 同一里程碑每局只触发一次。
- 触发后未选择前不继续触发新奖励。
- 奖励只在本局生效，结算后重置。
- 所有效果仍通过统一 modifiers 生效，UI 只展示选择，不参与计算。

## MVP 暂不做

- 排行榜。
- 完整广告商业化系统。
- 多人或社交竞争。
- 多角色。
- 皮肤和装扮商店。
- 长线每日任务系统。
- 复杂合成、材料、装备品质。

## MVP 验收

玩家 1 分钟内可以完成：

```text
开始下矿 -> 按住摇杆自由挖掘 -> 收集达标触发三选一 -> 氧气消耗 -> 捡矿
-> 回到地表出售 / 暂停返回结算 / 氧气耗尽
-> 结算金币 -> 升级 -> 再下一局
```
