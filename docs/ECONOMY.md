# 矿脉经济系统设计

## 设计前提

本项目面向抖音小游戏，不按传统长线 RPG 或买断制矿洞游戏设计经济。经济系统必须服务于短局循环、即时反馈和轻量成长。

| 抖音小游戏习惯 | 对经济设计的要求 |
|---|---|
| 单局时间短 | 30-90 秒内看到收益变化 |
| 竖屏轻操作 | 收益判断直观，不依赖复杂菜单 |
| 强反馈 | 挖到高价值矿、追到矿脉、结算增长要明显 |
| 低理解成本 | 矿物价值、背包压力、氧气风险一眼能懂 |
| 轻成长 | 每局都推动一点升级目标 |

## 核心目标

- 让玩家形成“再往下挖一点可能更赚”的冲动。
- 让玩家看到矿脉后产生“要不要横向追矿”的选择。
- 让背包、氧气、硬度、深度共同形成取舍。
- 让后续新增矿物主要改配置，不改核心采集和结算逻辑。

## 当前矿脉生成规则

第一版已从“按格移动/挖一格”改为“连续坐标地形”。玩家按住方向时，角色按输入方向、输入强度和持续时间移动；遇到非空气材质时，用圆形挖掘刷移除一段地形。

```text
玩家持续输入方向
  -> ContinuousRunManager 按 deltaTime 和输入强度计算目标坐标
  -> ContinuousTerrain 根据 seed + 连续坐标采样像素级地形材质
  -> DigBrushResolver 返回地形移除 delta、矿石 inventoryDelta、氧气消耗
  -> ContinuousRunManager 统一应用 terrain delta、inventory、oxygen、depth
  -> ContinuousTerrainView 以玩家为中心绘制附近连续材质
```

约束：

- `x/y` 是连续坐标，`y` 表示深度；允许向上、斜向和横向自由挖掘。
- 深度决定可出现矿物池，横向不能跳过深度成长。
- `富矿嗅觉` 只提升矿脉/背景矿概率，不直接改 UI 或结算。
- 高价值矿更多以小片矿脉出现，不满屏铺开。
- 每一局开始时重置连续地形和挖掘 delta，避免上一局挖空状态污染新局。

## 结算公式

第一版保持可解释：

```text
矿石收入 = sum(矿石数量 * 矿石单价)
深度基础奖励 = floor(最大深度 / 20) * 6
装备倍率 = 1 + 矿石手册等级加成
局内增益 = 收集里程碑触发三选一后获得的高价出货、深层奖金等 Roguelite 增益
最终收入 = floor((矿石收入 + 深度基础奖励) * 装备倍率) + 局内额外奖金
```

限制：

- 深度奖励只能是补充，不应超过主要矿石收入。
- 结算页必须展示收益来源，不能只显示总数。
- 装备倍率温和增长，避免数值膨胀过快。

## UI 展示

HUD 不展示全部矿物，避免挤爆顶部 20% UI 区。

HUD 显示：

```text
深度 / 氧气 / 背包 / 金币预估 / 当前增益 / 矿石总数与种类数
```

局内奖励提示：

```text
铜矿 x50 -> 矿工灵感三选一
选择后立即加入当前增益，并影响后续金币预估、背包或挖掘效率
```

结算页展示：

```text
铜矿 x8 = 40
铁矿 x3 = 33
金矿 x2 = 70
矿石压缩：节省 4 格背包
深度基础奖励：12
本局收入：155
```

## 已落地代码约定

| 能力 | 落点 |
|---|---|
| 矿物类型 | `OreType`、`RunInventory` |
| 矿物配置 | `assets/scripts/config/GameConfig.ts` 的 `ORE_TYPES`、`ORE_DEPTH_CONFIG`、`TILE_CONFIG` |
| 连续地形配置 | `assets/scripts/gameplay/terrain/TerrainConfig.ts` |
| 采集结算 | `DigBrushResolver` 返回地形和矿物 delta，`ContinuousRunManager` 统一写入状态 |
| 矿洞生成 | `ContinuousTerrain` 按连续坐标确定性采样地形材质 |
| 视口显示 | `ContinuousTerrainView` 以玩家为中心绘制附近连续地形 |
| 局内奖励 | `ContinuousRunRewardTracker` 按收集里程碑触发三选一 |
| 金币结算 | `ContinuousRunEconomy` 按 `ORE_TYPES` 自动生成矿石收益明细 |
| HUD | 只显示矿石总数和种类数 |
| 结算页 | 展示矿石收益、深度奖励、倍率、局内增益和压缩节省 |

## 暂不做

- 市场价格每日波动。
- 多货币系统。
- 矿物合成。
- 复杂装备词条。
- 离线收益。
- 广告翻倍作为核心经济。
- 强制分享或强社交任务。
