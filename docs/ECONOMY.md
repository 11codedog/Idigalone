# 矿脉经济系统设计

## 设计前提

本项目面向抖音小游戏，不按传统长线 RPG 或买断制矿洞游戏设计经济。经济系统必须服务于短局循环、即时反馈和轻量成长。

| 抖音小游戏习惯 | 对经济设计的要求 |
|---|---|
| 单局时间短 | 30-90 秒内看到收益变化 |
| 竖屏轻操作 | 收益判断直观，不依赖复杂菜单 |
| 强反馈 | 挖到高价值矿、连成矿脉、结算增长要明显 |
| 低理解成本 | 矿物价值、背包压力、氧气风险一眼能懂 |
| 轻成长 | 每局都推动一点升级目标 |
| 广告后置 | 激励视频只做失败补救、收益追加或体力补充，不影响 MVP 核心闭环 |

## 核心目标

- 让玩家形成“再往下挖一点可能更赚”的冲动。
- 让矿石不再是完全散点，而是能看出矿脉聚集。
- 让背包、氧气、硬度、深度共同形成取舍。
- 让后续新增矿物主要改配置，不改核心生成、采集和结算逻辑。

## 第一版实装范围

第一轮只实装 6 种矿物，避免 UI、背包、结算和资源一次性膨胀。

| 矿物 | 最低深度 | 基础价值 | 背包占用 | 硬度 | 体验定位 |
|---|---:|---:|---:|---:|---|
| 铜矿 | 1m | 5 | 1 | 1 | 新手稳定收益 |
| 铁矿 | 24m | 11 | 1 | 2 | 中浅层过渡收益 |
| 银矿 | 40m | 18 | 1 | 2 | 中层惊喜收益 |
| 金矿 | 65m | 35 | 2 | 3 | 高价值但占背包 |
| 水晶 | 95m | 70 | 2 | 3 | 富矿层目标 |
| 黑曜矿 | 155m | 150 | 3 | 4 | 深层高风险矿 |

暂时预留但不实装：煤矿、锡矿、蓝宝石、红宝石、秘银、铂金、星陨矿、古代核心。

## 深度层级

| 层级 | 深度 | 主要矿物 | 体验目标 |
|---|---:|---|---|
| 地表浅层 | 1-30m | 铜矿、铁矿 | 教学、安全、稳定小收益 |
| 稳定矿层 | 31-80m | 铁矿、银矿、金矿 | 路线选择和背包压力开始出现 |
| 富矿层 | 81-160m | 金矿、水晶、黑曜矿前置 | 收益明显提升，氧气压力变强 |
| 危险矿层 | 161m+ | 黑曜矿 | 高硬度高价值，要求装备成长 |

## 矿脉生成规则

当前代码采用“按深度行缓存矿脉”的轻量方案，适合小游戏原型：

```text
生成某一深度行
  -> 判断这一行是否触发矿脉
  -> 如果触发，按深度矿物池选择矿物类型
  -> 随机矿脉中心 x 和半径
  -> 越靠近中心，同类矿概率越高
  -> 边缘混入泥土、石头、氧气包和少量背景矿
  -> 缓存结果，后续查询不重复随机
```

约束：
- 深度决定可出现矿物池。
- `富矿嗅觉` 只提升矿脉/背景矿概率，不直接改 UI 或结算。
- 高价值矿更多以小片矿脉出现，不满屏铺开。
- 每一局开始时重置矿洞，避免上一局挖空状态污染新局。

## 结算公式

第一版保持可解释：

```text
矿石收入 = sum(矿石数量 * 矿石单价)
深度基础奖励 = floor(最大深度 / 20) * 6
装备倍率 = 1 + 矿石手册等级加成
局内增益 = 高价出货、深层奖金等 Roguelite 增益
最终收入 = floor((矿石收入 + 深度基础奖励) * 装备倍率) + 局内额外奖金
```

限制：
- 深度奖励只能是补充，不应超过主要矿石收入。
- 结算页必须展示收益来源，不能只显示总数。
- 装备倍率温和增长，避免数值膨胀过快。

## UI 展示

HUD 不展示全部矿物，避免挤爆顶部 20% UI 区。

HUD 显示：
```text
深度 / 氧气 / 背包 / 金币预估 / 当前增益 / 矿石总数与种类数
```

结算页展示：
```text
铜矿 x8 = 40
铁矿 x3 = 33
金矿 x2 = 70
深度基础奖励：12
本局收入：155
```

## 已落地代码约定

| 能力 | 落点 |
|---|---|
| 矿物类型 | `OreType`、`RunInventory` |
| 矿物配置 | `assets/scripts/config/GameConfig.ts` 的 `ORE_TYPES`、`ORE_DEPTH_CONFIG`、`TILE_CONFIG` |
| 采集结算 | `TileEffectResolver` 返回 `inventoryDelta`，`RunManager` 统一写入状态 |
| 矿洞生成 | `MineGrid` 按深度矿物池和按行矿脉生成 |
| HUD | 只显示矿石总数和种类数 |
| 结算页 | 按 `ORE_TYPES` 自动生成矿石收益明细 |

## 暂不做

- 市场价格每日波动。
- 多货币系统。
- 矿物合成。
- 复杂装备词条。
- 离线收益。
- 广告翻倍作为核心经济。
- 强制分享或强社交任务。
